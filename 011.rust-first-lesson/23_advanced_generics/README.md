泛型参数，它有三种常见的使用场景：

- 使用泛型参数延迟数据结构的绑定；
- 使用泛型参数和 PhantomData，声明数据结构中不直接使用，但在实现过程中需要用到的类型；
- 使用泛型参数让同一个数据结构对同一个 trait 可以拥有不同的实现。
